# Simulation Workspace

Use this folder for:

- experiment notes
- benchmark CSVs and processed summaries
- scenario definitions
- parameter sweeps
- pilot-comparison exports

## Suggested structure

- `inputs/` scenario definitions, occupancy maps, or cost-map examples
- `outputs/` processed metrics and charts
- `notebooks/` exploratory analysis
- `notes/` experiment logs

Keep raw benchmark claims traceable to a scenario, parameter set, and output artifact.
